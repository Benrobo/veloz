## Getting Started

> Make sure you have the latest version of [**Node.js**](https://nodejs.org/en/) installed.

<Steps>

### Step 1

**Install Veloz ClI**

```bash
# npm
npm install -g veeloz

# yarn
yarn global add veeloz
```

### Step 2

After installation of _veeloz_ cli, you would need to authenticate the cli with your Veeloz account token. You can get your token from [here](https://tryveeloz.com/settings).

![veeloz-settings](/images/screenshots/v-settings.png)

Run the command below

```bash
veeloz auth
```

This would work you through the steps of authenticating your cli with your veloz account.

</Steps>
