# Lemonsqueezy API Key

Lemonsqueezy uses `API_KEY` in authenticating users request from your app to their server.

## Creating API Key

Navigate to your lemonsqueezy account dashboard. Click the dropdown sidebar item called `Settings`, scroll down till you find `API`, then click on it.

Once created, update your `.env` variables with your api key.

```bash .env
LEMONSQUEEZY_API_KEY="xxx.xxx.xxx"
```
